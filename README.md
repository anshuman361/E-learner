# 🎓 E-Learner

A modern AI-powered online learning marketplace built with Next.js, inspired by platforms like Udemy. E-Learner enables instructors to create and manage courses while allowing students to learn through videos, articles, quizzes, AI assistance, and real-time communication.

## 🚀 Live Demo

**Website:** https://elearner-pro.vercel.app/
**GitHub Repository:** https://github.com/anshuman361/E-learner

---

# 📖 Overview

E-Learner is a full-stack EdTech platform where instructors can publish courses and interact with students, while students can enroll in courses, track progress, ask doubts, generate quizzes, and receive AI-powered learning assistance.
The project focuses on real-world software engineering concepts such as authentication, role-based access control, real-time communication, AI integration, cloud media management, and scalable backend architecture.

# ✨ Features

## 👨‍🎓 Student Features

- AI-Assistant powered doubt solving
- AI-assistant generated lesson summaries
- AI-assistant to take quizzes for self-assessment
- Secure authentication using Privy
- Browse available courses
- Enroll in courses
- Watch video lessons
- Read article lessons
- Chat with instructors in real time

## 👨‍🏫 Instructor Features

- Dashboard for course management
- Create and publish courses
- Secure authentication using Privy
- Real-time chat with students using Socket.io
- Upload video lessons & article
- Organize content into Chapters and Sections
